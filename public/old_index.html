Welcome to Nodester!

We do not currently have a full featured website as we are focusing on the API and the service. We are currently running the "stable" version of Node v.0.4.0 and we support Node.JS sandboxing of your apps which means that you can install your own NPM modules using our APPNPM API! Git is required to push your updates to Nodester. All we ask is that you be a good steward of our service and community.

Check out our new Command Line Interface (CLI)!

  1. npm install nodester-cli
  2. type "nodester" to see a list of command uses

Request a coupon via CLI or cURL to get on our invite list!

nodester coupon your_address@gmail.com

- OR -

curl -X POST -d "email=your_address@gmail.com" http://nodester.com/coupon

Deployment is Easy - Create, Git Push, & Run!

Remember to update your application's port address with the port address that we assign your application when you create it in step one.

nodester app create myapp hello.js

git remote add nodester the_url_returned_by_our_api and git push nodester master

NPM modules? nodester appnpm install express

nodester app start myapp

- OR -

curl -X POST -u "testuser:123" -d "appname=myapp&start=hello.js" http://api.nodester.com/app

git remote add nodester the_url_returned_by_our_api and git push nodester master

NPM modules? curl -X POST -u "testuser:123" -d "appname=myapp&action=install&package=express" http://api.nodester.com/appnpm

curl -X PUT -u "testuser:123" -d "appname=myapp&running=true" http://api.nodester.com/app Visit your app at http://myapp.nodester.com

Don't worry :) You only need to start the application once. Subsequent git pushes will automatically pick up your changes and restart.
